# RFID with STM32F103C8T6

**Connection Pins**
- VCC ---------- 3.3V
- GND ---------- GND
- RST ----------- 3.3V
- CS (SDA) ------- PA4
- SCK ----------- PA5
- MISO ---------- PA6
- MOSI ---------- PA7
- IRQ ------------ ---

![](https://os.mbed.com/media/uploads/hudakz/stm32f103c8t6_pinout_voltage01.png)

**Software Development Tools**
- STM32CubeMx
- Keil uVision
- STM Studio
